## 🥊 About

Trishool MMA is a modern, high-performance website for a Mixed Martial Arts academy based in Dehradun, Uttarakhand. The website features stunning animations, smooth transitions, and a bold color scheme perfect for showcasing combat sports training programs.

### ✨ Features

- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Smooth Animations** - Eye-catching scroll animations and transitions using AOS library
- **Modern UI/UX** - Clean, professional design with intuitive navigation
- **Interactive Elements** - Hover effects, parallax scrolling, and animated counters
- **SEO Optimized** - Proper meta tags and semantic HTML structure
- **Fast Loading** - Optimized assets and efficient code
- **Contact Form** - Integrated contact form for lead generation
- **Google Maps Integration** - Interactive map showing academy location

---

## 🎨 Design Highlights

- **Color Scheme**: Bold red (#ff0000) primary with gold (#ffd700) accents on dark backgrounds
- **Typography**: Bebas Neue for headings, Poppins for body text, Oswald for subtitles

## 🚀 Quick Start

### Option 1: View Locally

1. Clone or download this repository
2. Open `index.html` in your web browser
3. That's it! No build process required.

### Option 2: Deploy to GitHub Pages

1. **Initialize Git** (if not already done):
```bash
git init
git add .
git commit -m "Initial commit"
```

2. **Push to GitHub**:
```bash
git branch -M main
git remote add origin https://github.com/AbhixGupta/Trishool.git
git push -u origin main
```

3. **Enable GitHub Pages**:
   - Go to your repository settings
   - Navigate to "Pages" section
   - Select "main" branch as source
   - Save

Your site will be live at: `https://abhixgupta.github.io/Trishool/`

## 🔧 Customization

### Changing Colors

Edit the CSS variables in `assets/css/style.css`:

```css
:root {
    --primary-color: #ff0000;      /* Main red color */
    --secondary-color: #1a1a1a;    /* Dark backgrounds */
    --accent-color: #ffd700;       /* Gold accents */
    --text-color: #ffffff;         /* White text */
}
```

### Adding Images

Replace placeholder divs in HTML with actual images:

```html
<!-- Before -->
<div class="image-placeholder">
    <i class="fas fa-images"></i>
</div>

<!-- After -->
<img src="assets/images/your-image.jpg" alt="Description">
```

### Updating Content

All content can be edited directly in `index.html`. Sections are clearly marked with HTML comments.
